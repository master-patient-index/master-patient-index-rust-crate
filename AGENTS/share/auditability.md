### Auditability

**Event Streaming Publishing**: Automatic events for all changes

- \*Created, \*Updated, \*Deleted
- \*Merged, \*Linked, \*Unlinked

**Audit Logging**: Complete audit trail in PostgreSQL

- Old/new values as JSON
- User tracking (user_id, user_ip_address, user_agent)
- Timestamp-based audit history

**Audit Query API**: RESTful endpoints for audit log access

- Get audit history
- Get recent system-wide audits
- Get user-specific audit logs
